# IA illimitée, un prix fixe

Claudin.io est un proxy API pour les agents de codage IA. Payez un **abonnement mensuel fixe** et utilisez-le autant que vous le souhaitez — pas de facturation par token, pas de compteurs de requêtes, pas de crédits qui expirent.

## Pourquoi Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __Utilisation illimitée__

    ---

    Pas de compteurs de requêtes, pas de comptage de tokens. La seule limite est un plafond de **protection des dépenses** par période qui arrête un agent incontrôlable — vous ne l'atteignez presque jamais en utilisation normale.

-   :material-power-plug:{ .lg .middle } __Fonctionne avec vos outils__

    ---

    Des endpoints compatibles OpenAI et Anthropic. Intégrez-le dans OpenCode, Claude Code, Kilo Code, Zed, Codex, Cursor, ou tout client OpenAI en quelques minutes.

-   :material-cash-multiple:{ .lg .middle } __Coût prévisible__

    ---

    Un abonnement fixe. Pas de factures surprises à la fin du mois, pas d'anxiété du genre « combien va coûter cette refactorisation ? »

-   :material-shield-check:{ .lg .middle } __Protection des dépenses intégrée__

    ---

    Un plafond budgétaire en cascade par période vous protège des agents en boucle infinie qui épuisent votre forfait. Transparent et affiché en direct dans votre tableau de bord.

</div>

## Commencez en 4 étapes

1. **[Créez un compte](getting-started/account.md)** et copiez votre clé API.
2. **[Définissez votre clé API](getting-started/set-your-key.md)** une fois dans votre shell.
3. **[Connectez votre outil](clients/opencode.md)** — choisissez votre éditeur ou agent.
4. **[Effectuez votre premier appel](getting-started/first-call.md)** et commencez à construire.

!!! tip "Le nom du modèle est toujours `claudinio`"
    Quel que soit le client que vous utilisez, l'ID du modèle que vous ciblez est **`claudinio`**. L'URL de base est **`https://api.claudin.io`**.
