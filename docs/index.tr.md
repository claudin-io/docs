# Sınırsız AI, tek sabit fiyat

Claudin.io, AI kodlama ajanları için bir API proxy'sidir. **Sabit aylık
abonelik** ödeyin ve istediğiniz kadar kullanın — token başına ücretlendirme yok,
istek sayacı yok, son kullanma tarihi olan kredi yok.

</div>

## Neden Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __Sınırsız kullanım__

    ---

    İstek sayısı yok, token ölçümü yok. Tek koruma, başıboş bir ajanı durduran
    dönem başına **harcama koruma** sınırıdır — normal çalışmada neredeyse hiç
    karşılaşmazsınız.

-   :material-power-plug:{ .lg .middle } __Araçlarınızla çalışır__

    ---

    OpenAI ve Anthropic uyumlu uç noktalar. Dakikalar içinde OpenCode, Claude
    Code, Kilo Code, Zed, Codex, Cursor veya herhangi bir OpenAI istemcisine
    ekleyin.

-   :material-cash-multiple:{ .lg .middle } __Öngörülebilir maliyet__

    ---

    Sabit bir abonelik. Ay sonunda sürpriz faturalar yok,
    "bu yeniden düzenleme ne kadar tutacak?" endişesi yok.

-   :material-shield-check:{ .lg .middle } __Dahili harcama koruması__

    ---

    Kademeli dönem başına bütçe sınırı, sonsuz döngü ajanlarının planınızı
    tüketmesini önler. Şeffaf ve panelinizde canlı olarak gösterilir.

</div>

## 4 adımda başlayın

1. **[Hesap oluşturun](getting-started/account.md)** ve API anahtarınızı kopyalayın.
2. **[API anahtarınızı ayarlayın](getting-started/set-your-key.md)** shell'inizde bir kez.
3. **[Aracınızı bağlayın](clients/opencode.md)** — editörünüzü veya ajanınızı seçin.
4. **[İlk çağrınızı yapın](getting-started/first-call.md)** ve oluşturmaya başlayın.

!!! tip "Model adı her zaman `claudinio`'dur"
    Hangi istemciyi kullanırsanız kullanın, işaret ettiğiniz model kimliği
    **`claudinio`**'dur. Temel URL **`https://api.claudin.io`**'dir.
