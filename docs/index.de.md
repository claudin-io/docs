# Unbegrenzte KI, ein Festpreis

Claudin.io ist ein API-Proxy für KI-Codierungsagenten. Zahlen Sie ein **pauschales monatliches Abonnement** und nutzen Sie es so viel Sie wollen – keine token-basierte Abrechnung, keine Anfragenzähler, keine Guthaben, die verfallen.

## Warum Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __Unbegrenzte Nutzung__

    ---

    Keine Anfragenzähler, keine Token-Messung. Die einzige Einschränkung ist ein
    pro Zeitraum gestaffelter **Ausgabenschutz**, der einen durchgehenden Agenten
    stoppt – unter normaler Arbeit erreichen Sie ihn fast nie.

-   :material-power-plug:{ .lg .middle } __Funktioniert mit Ihren Tools__

    ---

    OpenAI- und Anthropic-kompatible Endpunkte. Fügen Sie es in OpenCode, Claude
    Code, Kilo Code, Zed, Codex, Cursor oder einen beliebigen OpenAI-Client
    in Minuten ein.

-   :material-cash-multiple:{ .lg .middle } __Vorhersehbare Kosten__

    ---

    Ein festes Abonnement. Keine Überraschungsrechnungen am Monatsende, keine
    Angst vor „Wie viel wird diese Umstrukturierung kosten?“

-   :material-shield-check:{ .lg .middle } __Ausgabenschutz integriert__

    ---

    Ein kaskadierendes Budgetlimit pro Zeitraum schützt Sie vor
    Endlosschleifen-Agenten, die Ihr Abonnement verbrauchen. Transparent und
    live in Ihrem Dashboard sichtbar.

</div>

## Loslegen in 4 Schritten

1. **[Erstellen Sie ein Konto](getting-started/account.md)** und kopieren Sie Ihren API-Schlüssel.
2. **[Setzen Sie Ihren API-Schlüssel](getting-started/set-your-key.md)** einmalig in Ihrer Shell.
3. **[Verbinden Sie Ihr Tool](clients/opencode.md)** – wählen Sie Ihren Editor oder Agenten.
4. **[Tätigen Sie Ihren ersten Aufruf](getting-started/first-call.md)** und beginnen Sie mit dem Bauen.

!!! tip "Der Modellname ist immer `claudinio`"
    Egal welchen Client Sie verwenden, die Modell-ID, auf die Sie verweisen, ist **`claudinio`**.
    Die Basis-URL ist **`https://api.claudin.io`**.
