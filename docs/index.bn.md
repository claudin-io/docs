# অসীম AI, একটি নির্দিষ্ট মূল্য

Claudin.io হল AI কোডিং এজেন্টদের জন্য একটি API প্রক্সি। একটি **ফ্ল্যাট মাসিক সাবস্ক্রিপশন** দিন এবং যত খুশি ব্যবহার করুন — কোনো প্রতি-টোকেন বিলিং নেই, কোনো অনুরোধ কাউন্টার নেই, কোনো ক্রেডিট নেই যা শেষ হয়ে যায়।

## কেন Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __অসীম ব্যবহার__

    ---

    কোনো অনুরোধ গণনা নেই, কোনো টোকেন মিটারিং নেই। একমাত্র সুরক্ষা হলো প্রতি-সময়কাল **ব্যয় সুরক্ষা** ক্যাপ যা একটি উন্মুক্ত এজেন্ট বন্ধ করে — সাধারণ কাজে আপনি এটি প্রায় কখনই আঘাত করেন না।

-   :material-power-plug:{ .lg .middle } __আপনার টুলসের সাথে কাজ করে__

    ---

    OpenAI এবং Anthropic-সামঞ্জস্যপূর্ণ এন্ডপয়েন্ট। এটি মিনিটের মধ্যে OpenCode, Claude Code, Kilo Code, Zed, Codex, Cursor বা যেকোনো OpenAI ক্লায়েন্টে ড্রপ করুন।

-   :material-cash-multiple:{ .lg .middle } __পূর্বানুমানযোগ্য খরচ__

    ---

    একটি নির্দিষ্ট সাবস্ক্রিপশন। মাসের শেষে কোনো অপ্রত্যাশিত বিল নেই, 'এই রিফ্যাক্টরের খরচ কত হবে?' এই উদ্বেগ নেই।

-   :material-shield-check:{ .lg .middle } __ব্যয় সুরক্ষা বিল্ট-ইন__

    ---

    একটি ক্যাসকেডিং প্রতি-সময়কাল বাজেট ক্যাপ আপনাকে অসীম-লুপ এজেন্ট থেকে রক্ষা করে যা আপনার প্ল্যান পুড়িয়ে দেয়। স্বচ্ছ এবং আপনার ড্যাশবোর্ডে লাইভ দেখানো হয়।

## ৪ ধাপে শুরু করুন

1. **[একটি অ্যাকাউন্ট তৈরি করুন](getting-started/account.md)** এবং আপনার API কী কপি করুন।
2. **[আপনার API কী সেট করুন](getting-started/set-your-key.md)** শেলটিতে একবার।
3. **[আপনার টুল সংযুক্ত করুন](clients/opencode.md)** — আপনার এডিটর বা এজেন্ট বেছে নিন।
4. **[আপনার প্রথম কল করুন](getting-started/first-call.md)** এবং বিল্ডিং শুরু করুন।

!!! tip "মডেলের নাম সবসময় `claudinio`"
    আপনি যেকোনো ক্লায়েন্ট ব্যবহার করুন, আপনি যে মডেল আইডি নির্দেশ করেন তা হল **`claudinio`**। বেস URL হল **`https://api.claudin.io`**।
