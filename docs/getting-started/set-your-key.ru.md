# Установите ваш API-ключ

Установите ваш ключ Claudin.io **один раз** как переменную окружения, и каждый инструмент
из этого руководства сможет его использовать — не нужно вставлять его вручную в каждый клиент.

Возьмите ваш ключ `sk-...` из [панели управления](https://claudin.io/dashboard) (смотрите
[Создание учетной записи](account.md)), затем добавьте его в ваш профиль оболочки, чтобы он был
доступен в каждом новом терминале.

## macOS / Linux

=== "zsh (по умолчанию в macOS)"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.zshrc
    source ~/.zshrc
    ```

=== "bash"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.bashrc
    source ~/.bashrc
    ```

Замените `sk-...` на ваш реальный ключ. Не уверены, какая оболочка используется? Выполните
`echo $SHELL`.

## Проверка

```bash
echo $CLAUDINIO_API_KEY
```

Вы должны увидеть напечатанный ключ. Если он пуст, откройте новый терминал или
повторно выполните команду `source` выше.

## Зачем это нужно

Каждый скрипт **Быстрой настройки** в разделе [Подключение вашего инструмента](../clients/opencode.md)
читает `$CLAUDINIO_API_KEY`, поэтому после его экспорта вы можете запустить любой из них
без изменений — не нужно заменять `YOUR_API_KEY`. Инструменты, которые читают переменные
окружения напрямую (Codex's `env_key`, любой OpenAI-совместимый CLI), также подхватывают его
автоматически.

!!! warning "Относитесь к ключу как к паролю"
    Любой, у кого есть этот ключ, может расходовать бюджет вашего плана. Не коммитьте ваш
    `~/.zshrc` / `~/.bashrc` в публичный репозиторий. Если ключ утек, отзовите его в
    панели управления и экспортируйте новый.

---

Ключ экспортирован? Теперь [совершите первый вызов](first-call.md) или перейдите сразу к
[подключению вашего инструмента](../clients/opencode.md).
