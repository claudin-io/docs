# アカウントを作成する

動作するAPIキーの取得は約1分で完了します。

## 1. GitHubでサインイン

**[claudin.io](https://claudin.io)** にアクセスし、**Sign in with GitHub** をクリックします。
Claudin.ioはログインにGitHubを使用します。別途パスワードを管理する必要はありません。

初回サインイン時に、アカウントは自動的に **Free** プランで作成されます。料金を支払う前に試すことができます。

## 2. APIキーを生成する

[ダッシュボード](https://claudin.io/dashboard) に入ったら：

1. **API Keys** カードを見つけます。
2. **Generate key** (または **Create new key**) をクリックします。
3. キーをコピーします。`sk-...` のような形式です。

!!! warning "キーをパスワードのように扱ってください"
    APIキーはプランの予算にアクセスできるため、リポジトリにコミットしたり、公開チャットに貼り付けたり、共有したりしないでください。キーが漏洩した場合は、ダッシュボードからそれを無効化し、新しいキーを生成してください。

## 3. 必要な2つの値を確認する

すべての統合で同じ2つの値が必要です：

| 値 | 説明 |
| --- | --- |
| **Base URL** | `https://api.claudin.io` |
| **Model** | `claudinio` |
| **API key** | 先ほどコピーした `sk-...` |

以上です。次は、[生のAPI呼び出しを行う](first-call.md)か、または[ツールの接続](../clients/claude-code.md)に直接進んでください。

## プランの選択

**Free** プランのまま試すことができます。より多くの容量が必要になったら、ダッシュボードからアップグレードしてください。詳細は [プランと制限](../plans.md) を参照してください。

アップグレードはStripeを通じて処理され、即座に反映されます。
