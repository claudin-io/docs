# अपना खाता बनाएं

काम करने वाली API कुंजी प्राप्त करने में लगभग एक मिनट लगता है।

## 1. GitHub से साइन इन करें

**[claudin.io](https://claudin.io)** पर जाएं और **Sign in with GitHub** पर क्लिक करें।
Claudin.io लॉगिन के लिए GitHub का उपयोग करता है — इसे अलग से पासवर्ड प्रबंधित करने की आवश्यकता नहीं है।

पहली बार साइन इन करने पर, आपका खाता स्वचालित रूप से **Free** योजना पर बनाया जाता है,
ताकि आप कुछ भी भुगतान करने से पहले इसे आज़मा सकें।

## 2. अपनी API कुंजी उत्पन्न करें

[dashboard](https://claudin.io/dashboard) में आने के बाद:

1. **API Keys** कार्ड ढूंढें।
2. **Generate key** (या **Create new key**) पर क्लिक करें।
3. कुंजी को कॉपी करें — यह `sk-...` जैसी दिखती है।

!!! warning "अपनी कुंजी को पासवर्ड की तरह समझें"
    आपकी API कुंजी आपकी योजना के बजट तक पहुंच प्रदान करती है। इसे किसी रिपॉजिटरी में कमिट न करें,
    सार्वजनिक चैट में पेस्ट न करें, या साझा न करें। यदि कोई कुंजी लीक हो जाती है, तो डैशबोर्ड से
    इसे रद्द करें और एक नई कुंजी उत्पन्न करें।

## 3. आपको जिन दो मानों की आवश्यकता होगी, उन्हें नोट करें

हर एकीकरण को समान दो चीजों की आवश्यकता होती है:

| मान | यह क्या है |
| --- | --- |
| **Base URL** | `https://api.claudin.io` |
| **Model** | `claudinio` |
| **API key** | वह `sk-...` जो आपने अभी-अभी कॉपी किया है |

बस इतना ही। इसके बाद, या तो [कच्चा API कॉल करें](first-call.md) यह पुष्टि करने के लिए कि यह काम करता है,
या सीधे [अपने टूल को कनेक्ट करें](../clients/claude-code.md) पर जाएं।

## योजना चुनना

आप चीजों को आज़माने के लिए **Free** पर रह सकते हैं। जब आप अधिक क्षमता के लिए तैयार हों,
तो डैशबोर्ड से अपग्रेड करें — पूर्ण विवरण के लिए [योजनाएं और सीमाएं](../plans.md) देखें।

अपग्रेड Stripe के माध्यम से संभाले जाते हैं और तुरंत प्रभावी होते हैं।
