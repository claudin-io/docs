# أنشئ حسابك

الحصول على مفتاح API فعال يستغرق حوالي دقيقة.

## 1. تسجيل الدخول باستخدام GitHub

اذهب إلى **[claudin.io](https://claudin.io)** وانقر على **تسجيل الدخول باستخدام GitHub**.
يستخدم Claudin.io GitHub لتسجيل الدخول — لا توجد كلمة مرور منفصلة لإدارتها.

في المرة الأولى التي تسجل فيها الدخول، يتم إنشاء حسابك تلقائيًا على الخطة **المجانية**، حتى تتمكن من تجربتها قبل دفع أي شيء.

## 2. إنشاء مفتاح API الخاص بك

بمجرد أن تكون في [لوحة التحكم](https://claudin.io/dashboard):

1. ابحث عن بطاقة **مفاتيح API**.
2. انقر على **إنشاء مفتاح** (أو **إنشاء مفتاح جديد**).
3. انسخ المفتاح — يبدو مثل `sk-...`.

!!! warning "عامل مفتاحك مثل كلمة المرور"
    يمنح مفتاح API الخاص بك الوصول إلى ميزانية خطتك. لا تلتزم به في مستودع، أو تلصقه في دردشة عامة، أو تشاركه. إذا تسرب مفتاح، قم بإبطاله من لوحة التحكم وإنشاء مفتاح جديد.

## 3. لاحظ القيمتين التي ستحتاجها

كل تكامل يحتاج إلى نفس الأمرين:

| القيمة | ما هي |
| --- | --- |
| **عنوان URL الأساسي** | `https://api.claudin.io` |
| **النموذج** | `claudinio` |
| **مفتاح API** | `sk-...` الذي نسخته للتو |

هذا كل شيء. بعد ذلك، إما [إجراء استدعاء API خام](first-call.md) لتأكيد أنه يعمل، أو الانتقال مباشرة إلى [توصيل أداتك](../clients/claude-code.md).

## اختيار الخطة

يمكنك البقاء على الخطة **المجانية** لتجربة الأشياء. عندما تكون مستعدًا لمزيد من السعة، قم بالترقية من لوحة التحكم — راجع [الخطط والحدود](../plans.md) للتفصيل الكامل.

تتم معالجة الترقيات عبر Stripe وتصبح سارية المفعول فورًا.
