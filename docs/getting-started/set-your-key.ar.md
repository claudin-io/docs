# تعيين مفتاح API الخاص بك

قم بتعيين مفتاح Claudin.io **مرة واحدة** كمتغير بيئة وكل أداة في هذا الدليل يمكنها إعادة استخدامه — لا حاجة للصقه في كل عميل يدويًا.

احصل على مفتاح `sk-...` الخاص بك من [لوحة التحكم](https://claudin.io/dashboard) (اطلع على [إنشاء حسابك](account.md))، ثم أضفه إلى ملف شيل الخاص بك ليكون متاحًا في كل طرفية جديدة.

## macOS / Linux

=== "zsh (الافتراضي على macOS)"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.zshrc
    source ~/.zshrc
    ```

=== "bash"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.bashrc
    source ~/.bashrc
    ```

استبدل `sk-...` بمفتاحك الحقيقي. لست متأكدًا من الشيل الذي تستخدمه؟ شغل `echo $SHELL`.

## التحقق

```bash
echo $CLAUDINIO_API_KEY
```

يجب أن ترى مفتاحك مطبوعًا. إذا كان فارغًا، افتح طرفية جديدة أو أعد تشغيل الأمر `source` أعلاه.

## لماذا يساعد هذا

كل نص **الإعداد السريع** في قسم [الاتصال بأداتك](../clients/opencode.md) يقرأ `$CLAUDINIO_API_KEY`، لذلك بمجرد تصديره يمكنك تشغيل أي منها كما هو — لا يوجد `YOUR_API_KEY` لاستبداله. الأدوات التي تقرأ متغيرات البيئة مباشرة (مثل `env_key` الخاص بـ Codex، وأي CLI متوافق مع OpenAI) تلتقطه تلقائيًا أيضًا.

!!! warning "عامل مفتاحك مثل كلمة المرور"
    أي شخص لديه هذا المفتاح يمكنه إنفاق ميزانية خطتك. لا تقم بإيداع ملفات `~/.zshrc` / `~/.bashrc` في مستودع عام. إذا تسرب المفتاح، قم بإبطاله في لوحة التحكم وقم بتصدير مفتاح جديد.

---

هل قمت بتصدير المفتاح؟ الآن [قم بإجراء أول اتصال](first-call.md) أو انتقل مباشرة إلى [توصيل أداتك](../clients/opencode.md).
