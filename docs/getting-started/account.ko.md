# 계정 만들기

작동하는 API 키를 얻는 데 약 1분이 걸립니다.

## 1. GitHub로 로그인

**[claudin.io](https://claudin.io)**로 이동하여 **GitHub로 로그인**을 클릭하세요.
Claudin.io는 로그인에 GitHub를 사용하므로 별도의 비밀번호를 관리할 필요가 없습니다.

처음 로그인하면 **무료** 요금제로 계정이 자동 생성되므로 비용을 지불하기 전에 사용해볼 수 있습니다.

## 2. API 키 생성

[대시보드](https://claudin.io/dashboard)에 접속한 후:

1. **API 키** 카드를 찾습니다.
2. **키 생성**(또는 **새 키 만들기**)을 클릭합니다.
3. 키를 복사합니다. 키는 `sk-...`와 같은 형태입니다.

!!! warning "키를 비밀번호처럼 취급하세요"
    API 키는 요금제의 예산에 접근할 수 있게 합니다. 리포지토리에 커밋하거나, 공개 채팅에 붙여넣거나, 공유하지 마세요. 키가 유출되면 대시보드에서 취소하고 새 키를 생성하세요.

## 3. 필요한 두 가지 값 확인

모든 통합에는 동일한 두 가지가 필요합니다.

| 값 | 설명 |
| --- | --- |
| **기본 URL** | `https://api.claudin.io` |
| **모델** | `claudinio` |
| **API 키** | 방금 복사한 `sk-...` |

이상입니다. 다음으로, [원시 API 호출](first-call.md)을 통해 작동을 확인하거나, 바로 [도구 연결](../clients/claude-code.md)로 넘어가세요.

## 요금제 선택

**무료** 요금제를 유지하며 시도해볼 수 있습니다. 더 많은 여유가 필요할 때는 대시보드에서 업그레이드하세요. 전체 내용은 [요금제 및 제한](../plans.md)을 참조하세요.

업그레이드는 Stripe을 통해 처리되며 즉시 적용됩니다.
