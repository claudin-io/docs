# আপনার অ্যাকাউন্ট তৈরি করুন

একটি কার্যকরী API কী পেতে প্রায় এক মিনিট সময় লাগে।

## ১. GitHub দিয়ে সাইন ইন করুন

**[claudin.io](https://claudin.io)**-এ যান এবং **Sign in with GitHub**-এ ক্লিক করুন।
Claudin.io লগইনের জন্য GitHub ব্যবহার করে — আলাদা পাসওয়ার্ড পরিচালনা করার প্রয়োজন নেই।

প্রথমবার সাইন ইন করার সময় আপনার অ্যাকাউন্ট স্বয়ংক্রিয়ভাবে **Free** প্ল্যানে তৈরি হয়, তাই কিছু দেওয়ার আগে আপনি এটি চেষ্টা করে দেখতে পারেন।

## ২. আপনার API কী তৈরি করুন

একবার আপনি [dashboard](https://claudin.io/dashboard)-এ গেলে:

1. **API Keys** কার্ডটি খুঁজুন।
2. **Generate key** (অথবা **Create new key**) ক্লিক করুন।
3. কীটি কপি করুন — এটি `sk-...` এর মতো দেখায়।

!!! warning "আপনার কীটি পাসওয়ার্ডের মতো আচরণ করুন"
    আপনার API কী আপনার প্ল্যানের বাজেটে অ্যাক্সেস দেয়। এটি রিপোজিটরিতে কমিট করবেন না, পাবলিক চ্যাটে পেস্ট করবেন না, বা শেয়ার করবেন না। যদি কোনো কী ফাঁস হয়, তাহলে ড্যাশবোর্ড থেকে এটি রিভোক করুন এবং একটি নতুন তৈরি করুন।

## ৩. আপনার প্রয়োজনীয় দুটি মান নোট করুন

প্রতিটি ইন্টিগ্রেশনের জন্য একই দুটি জিনিস প্রয়োজন:

| Value | What it is |
| --- | --- |
| **Base URL** | `https://api.claudin.io` |
| **Model** | `claudinio` |
| **API key** | আপনি সবেমাত্র কপি করা `sk-...` |

এটাই। এরপর, হয় [একটি কাঁচা API কল](first-call.md) করে এটি কাজ করে কিনা নিশ্চিত করুন, অথবা সরাসরি [আপনার টুল সংযোগ](../clients/claude-code.md)-এ চলে যান।

## একটি প্ল্যান নির্বাচন করা

আপনি জিনিসগুলি চেষ্টা করার জন্য **Free**-এ থাকতে পারেন। যখন আরও বেশি সুযোগের জন্য প্রস্তুত হন, ড্যাশবোর্ড থেকে আপগ্রেড করুন — সম্পূর্ণ বিবরণের জন্য [প্ল্যান ও সীমা](../plans.md) দেখুন।

আপগ্রেডগুলি Stripe-এর মাধ্যমে পরিচালিত হয় এবং তাৎক্ষণিকভাবে কার্যকর হয়।
