# اپنا اکاؤنٹ بنائیں

ایک کام کرنے والی API کلید حاصل کرنے میں تقریباً ایک منٹ لگتا ہے۔

## 1. GitHub سے سائن ان کریں

**[claudin.io](https://claudin.io)** پر جائیں اور **Sign in with GitHub** پر کلک کریں۔
Claudin.io لاگ ان کے لیے GitHub استعمال کرتا ہے — کوئی الگ پاس ورڈ سنبھالنے کی ضرورت نہیں۔

پہلی بار جب آپ سائن ان کرتے ہیں، آپ کا اکاؤنٹ خود بخود **Free** پلان پر بن جاتا ہے، تاکہ آپ کچھ بھی ادا کرنے سے پہلے اسے آزما سکیں۔

## 2. اپنی API کلید بنائیں

ایک بار جب آپ [ڈیش بورڈ](https://claudin.io/dashboard) میں آجائیں:

1. **API Keys** کارڈ تلاش کریں۔
2. **Generate key** (یا **Create new key**) پر کلک کریں۔
3. کلید کو کاپی کریں — یہ `sk-...` کی طرح نظر آتی ہے۔

!!! warning "اپنی کلید کو پاس ورڈ کی طرح رکھیں"
    آپ کی API کلید آپ کے پلان کے بجٹ تک رسائی فراہم کرتی ہے۔ اسے کسی ریپو میں کمٹ نہ کریں، عوامی چیٹ میں نہ چسپاں کریں، یا شیئر نہ کریں۔ اگر کلید لیک ہو جائے تو اسے ڈیش بورڈ سے منسوخ کریں اور ایک نئی بنائیں۔

## 3. ان دو اقدار کو نوٹ کریں جن کی آپ کو ضرورت ہوگی

ہر انضمام کو ایک ہی دو چیزوں کی ضرورت ہوتی ہے:

| قدر | یہ کیا ہے |
| --- | --- |
| **بنیادی URL** | `https://api.claudin.io` |
| **ماڈل** | `claudinio` |
| **API کلید** | وہ `sk-...` جو آپ نے ابھی کاپی کیا |

بس یہی ہے۔ اگلا، یا تو [خام API کال کریں](first-call.md) تاکہ تصدیق ہو کہ یہ کام کرتا ہے، یا سیدھے [اپنے ٹول کو جوڑنے](../clients/claude-code.md) پر جائیں۔

## پلان کا انتخاب

آپ چیزوں کو آزمانے کے لیے **Free** پر رہ سکتے ہیں۔ جب آپ مزید گنجائش کے لیے تیار ہوں تو ڈیش بورڈ سے اپ گریڈ کریں — مکمل تفصیل کے لیے [Plans & limits](../plans.md) دیکھیں۔

اپ گریڈز Stripe کے ذریعے سنبھالے جاتے ہیں اور فوری طور پر اثر انداز ہوتے ہیں۔
