# अपना API कुंजी सेट करें

अपनी Claudin.io कुंजी **एक बार** एक पर्यावरण चर के रूप में सेट करें और इस गाइड के सभी उपकरण इसका पुन: उपयोग कर सकते हैं — इसे प्रत्येक क्लाइंट में हाथ से पेस्ट करने की आवश्यकता नहीं है।

अपनी `sk-...` कुंजी [डैशबोर्ड](https://claudin.io/dashboard) से प्राप्त करें (देखें [अपना खाता बनाएं](account.md)), फिर इसे अपने शेल प्रोफ़ाइल में जोड़ें ताकि यह प्रत्येक नए टर्मिनल में उपलब्ध हो।

## macOS / Linux

=== "zsh (मैकोज़ पर डिफ़ॉल्ट)"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.zshrc
    source ~/.zshrc
    ```

=== "bash"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.bashrc
    source ~/.bashrc
    ```

`sk-...` को अपनी वास्तविक कुंजी से बदलें। पता नहीं कि आप किस शेल पर हैं? `echo $SHELL` चलाएँ।

## सत्यापित करें

```bash
echo $CLAUDINIO_API_KEY
```

आपको अपनी कुंजी वापस मुद्रित दिखनी चाहिए। यदि यह खाली है, तो एक नया टर्मिनल खोलें या ऊपर `source` कमांड को फिर से चलाएँ।

## यह क्यों मदद करता है

[Connect your tool](../clients/opencode.md) अनुभाग में प्रत्येक **त्वरित सेटअप** स्क्रिप्ट `$CLAUDINIO_API_KEY` को पढ़ती है, इसलिए एक बार निर्यात होने पर आप उनमें से किसी को भी यथावत चला सकते हैं — `YOUR_API_KEY` को बदलने की कोई आवश्यकता नहीं है। वे उपकरण जो सीधे पर्यावरण चर पढ़ते हैं (Codex का `env_key`, कोई भी OpenAI-संगत CLI) भी इसे स्वचालित रूप से उठा लेते हैं।

!!! warning "अपनी कुंजी को पासवर्ड की तरह समझें"
    इस कुंजी वाला कोई भी व्यक्ति आपकी योजना का बजट खर्च कर सकता है। अपने `~/.zshrc` / `~/.bashrc` को सार्वजनिक रिपॉजिटरी में कमिट न करें। यदि कुंजी लीक होती है, तो डैशबोर्ड में इसे रद्द करें और एक नई कुंजी निर्यात करें।
