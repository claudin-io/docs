# اپنی API کلید سیٹ کریں

اپنی Claudin.io کلید **ایک بار** بطور ماحولیاتی متغیر (environment variable) سیٹ کریں اور اس گائیڈ کا ہر ٹول اسے دوبارہ استعمال کر سکے گا — ہر کلائنٹ میں دستی طور پر چسپاں کرنے کی ضرورت نہیں۔

اپنی `sk-...` کلید [ڈیش بورڈ](https://claudin.io/dashboard) سے حاصل کریں (دیکھیں [اپنا اکاؤنٹ بنائیں](account.md))، پھر اسے اپنی شیل پروفائل میں شامل کریں تاکہ ہر نئے ٹرمینل میں دستیاب رہے۔

## macOS / Linux

=== "zsh (macOS میں ڈیفالٹ)"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.zshrc
    source ~/.zshrc
    ```

=== "bash"

    ```bash
    echo 'export CLAUDINIO_API_KEY="sk-..."' >> ~/.bashrc
    source ~/.bashrc
    ```

`sk-...` کو اپنی اصلی کلید سے بدلیں۔ پتہ نہیں آپ کون سی شیل استعمال کر رہے ہیں؟ `echo $SHELL` چلائیں۔

## تصدیق کریں

```bash
echo $CLAUDINIO_API_KEY
```

آپ کو اپنی کلید پرنٹ ہوتی دکھنی چاہیے۔ اگر خالی ہے تو نیا ٹرمینل کھولیں یا اوپر والا `source` کمانڈ دوبارہ چلائیں۔

## یہ کیوں مددگار ہے

[اپنا ٹول جوڑیں](../clients/opencode.md) سیکشن میں ہر **فوری سیٹ اپ** اسکرپٹ `$CLAUDINIO_API_KEY` پڑھتا ہے، لہٰذا ایک بار ایکسپورٹ ہونے کے بعد آپ ان سب کو بغیر کسی تبدیلی کے چلا سکتے ہیں — `YOUR_API_KEY` بدلنے کی ضرورت نہیں۔ جو ٹولز ماحولیاتی متغیرات براہِ راست پڑھتے ہیں (Codex کا `env_key`، کوئی بھی OpenAI-مطابق CLI) وہ بھی خود بخود اسے استعمال کر لیتے ہیں۔

!!! warning "اپنی کلید کو پاس ورڈ کی طرح سمجھیں"
    اس کلید کے ساتھ کوئی بھی آپ کے پلان کا بجٹ خرچ کر سکتا ہے۔ اپنے `~/.zshrc` / `~/.bashrc` کو عوامی ریپوزٹری میں اپ لوڈ نہ کریں۔ اگر کلید لیک ہو جائے تو ڈیش بورڈ میں اسے منسوخ کریں اور نئی ایکسپورٹ کریں۔

---

کلید ایکسپورٹ کر لی؟ اب [اپنی پہلی کال کریں](first-call.md) یا براہِ راست [اپنا ٹول جوڑیں](../clients/opencode.md) پر جائیں۔
