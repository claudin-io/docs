# ذكاء اصطناعي غير محدود، سعر ثابت واحد

Claudin.io هو وكيل API لوكلاء برمجة الذكاء الاصطناعي. ادفع **اشتراكًا شهريًا ثابتًا** واستخدمه بقدر ما تريد — لا فاتورة لكل رمز، ولا عدادات للطلبات، ولا أرصدة تنتهي صلاحيتها.

## لماذا Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __استخدام غير محدود__

    ---

    لا عدادات للطلبات، ولا قياس للرموز. الدرع الوحيد هو حد **حماية الإنفاق** لكل فترة يوقف الوكيل الجامح — نادرًا ما تصل إليه في العمل العادي.

-   :material-power-plug:{ .lg .middle } __يعمل مع أدواتك__

    ---

    نقاط نهاية متوافقة مع OpenAI وAnthropic. أدرجه في OpenCode أو Claude Code أو Kilo Code أو Zed أو Codex أو Cursor أو أي عميل OpenAI في دقائق.

-   :material-cash-multiple:{ .lg .middle } __تكلفة متوقعة__

    ---

    اشتراك ثابت. لا فواتير مفاجئة في نهاية الشهر، ولا قلق من "كم ستكلف هذه إعادة الهيكلة؟"

-   :material-shield-check:{ .lg .middle } __حماية الإنفاق مدمجة__

    ---

    حد ميزانية متدرج لكل فترة يحميك من الوكلاء في حلقات لا نهائية يحرقون خطتك. شفاف ويظهر مباشر في لوحة التحكم الخاصة بك.

</div>

## ابدأ في 4 خطوات

1. **[إنشاء حساب](getting-started/account.md)** وانسخ مفتاح API الخاص بك.
2. **[عيّن مفتاح API الخاص بك](getting-started/set-your-key.md)** مرة واحدة في شلّتك.
3. **[وصّل أداتك](clients/opencode.md)** — اختر المحرر أو الوكيل الخاص بك.
4. **[قم بأول استدعاء لك](getting-started/first-call.md)** وابدأ البناء.

!!! tip "اسم النموذج دائمًا هو `claudinio`"
    أيًا كان العميل الذي تستخدمه، فإن معرف النموذج الذي توجهه إليه هو **`claudinio`**. عنوان URL الأساسي هو **`https://api.claudin.io`**.
