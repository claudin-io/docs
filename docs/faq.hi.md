# FAQ

## Claudin.io वास्तव में क्या है?

Claudin.io AI कोडिंग एजेंटों के लिए एक API प्रॉक्सी है। आप एक फ्लैट मासिक सब्सक्रिप्शन का भुगतान करते हैं और एक OpenAI/Anthropic-संगत API कुंजी प्राप्त करते हैं जिसे आप Claude Code, Kilo, Zed, Codex, Cursor, या किसी भी OpenAI क्लाइंट में उपयोग कर सकते हैं। कोई प्रति-टोकन बिलिंग नहीं।

## क्या यह वास्तव में असीमित है?

उपयोग असीमित है — कोई अनुरोध काउंटर या टोकन मीटर नहीं है। एकमात्र सीमा प्रति समय विंडो एक **व्यय-सुरक्षा कैप** है जो एक भगोड़े एजेंट को आपकी योजना खत्म करने से रोकती है। सामान्य इंटरैक्टिव कार्य में आप शायद ही कभी इसे हिट करते हैं। देखें [योजनाएँ और सीमाएँ](plans.md)।

## मैं किस मॉडल का उपयोग करूँ?

हमेशा **`claudinio`** (या `claudinio/claudinio` उन क्लाइंट्स के लिए जो `provider/model` फॉर्म चाहते हैं)। बेस URL है `https://api.claudin.io`।

## क्या मैं `Authorization` या `x-api-key` से प्रमाणित करूँ?

दोनों काम करते हैं। `Authorization: Bearer YOUR_API_KEY` या `x-api-key: YOUR_API_KEY`।

## क्या मैं इसे किसी ऐसे टूल के साथ उपयोग कर सकता हूँ जो सूचीबद्ध नहीं है?

हाँ — कोई भी टूल जो आपको कस्टम OpenAI बेस URL सेट करने देता है, काम करता है। [सामान्य OpenAI सेटअप](clients/openai-compatible.md) का उपयोग करें।

## क्या यह टूल / फ़ंक्शन कॉलिंग का समर्थन करता है?

हाँ। यही कारण है कि यह एजेंटिक एडिटर्स के अंदर काम करता है। `tools` पास करें और `tool_calls` पढ़ें जैसे OpenAI API के साथ।

## क्या यह इमेज, ऑडियो या वीडियो संभाल सकता है?

हाँ, पारदर्शी रूप से। मानक OpenAI कंटेंट ब्लॉक भेजें; प्रॉक्सी इमेज/ऑडियो/वीडियो को टेक्स्ट विवरण या ट्रांसक्रिप्शन में बदल देता है इससे पहले कि मॉडल उन्हें देखे। कॉन्फ़िगर करने के लिए कुछ खास नहीं।
