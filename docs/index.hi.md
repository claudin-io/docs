# असीमित AI, एक निश्चित मूल्य

Claudin.io AI कोडिंग एजेंटों के लिए एक API प्रॉक्सी है। एक **फ्लैट मासिक सब्सक्रिप्शन** का भुगतान करें और जितना चाहें उपयोग करें — प्रति-टोकन बिलिंग नहीं, कोई अनुरोध काउंटर नहीं, कोई क्रेडिट जो समाप्त हो जाए।

## क्यों Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __असीमित उपयोग__

    ---

    कोई अनुरोध गणना नहीं, कोई टोकन मीटरिंग नहीं। एकमात्र सुरक्षा अवरोध प्रति-अवधि **व्यय सुरक्षा** कैप है जो एक अनियंत्रित एजेंट को रोकता है — सामान्य कार्य में आप लगभग कभी इससे नहीं टकराते।

-   :material-power-plug:{ .lg .middle } __आपके टूल्स के साथ काम करता है__

    ---

    OpenAI और Anthropic-संगत एंडपॉइंट्स। इसे OpenCode, Claude Code, Kilo Code, Zed, Codex, Cursor, या किसी भी OpenAI क्लाइंट में मिनटों में डालें।

-   :material-cash-multiple:{ .lg .middle } __पूर्वानुमानित लागत__

    ---

    एक निश्चित सब्सक्रिप्शन। महीने के अंत में कोई आश्चर्यजनक बिल नहीं, कोई "इस रिफैक्टर की लागत कितनी होगी?" की चिंता नहीं।

-   :material-shield-check:{ .lg .middle } __व्यय सुरक्षा निर्मित__

    ---

    एक कैस्केडिंग प्रति-अवधि बजट कैप आपको अनंत-लूप एजेंटों द्वारा आपकी योजना को जलाने से बचाता है। पारदर्शी और आपके डैशबोर्ड में लाइव दिखाया गया।

</div>

## 4 चरणों में शुरू करें

1. **[एक खाता बनाएं](getting-started/account.md)** और अपनी API कुंजी कॉपी करें।
2. **[अपनी API कुंजी सेट करें](getting-started/set-your-key.md)** एक बार अपने शेल में।
3. **[अपना टूल कनेक्ट करें](clients/opencode.md)** — अपना एडिटर या एजेंट चुनें।
4. **[अपनी पहली कॉल करें](getting-started/first-call.md)** और बिल्डिंग शुरू करें।

!!! tip "मॉडल का नाम हमेशा `claudinio` है"
    आप जो भी क्लाइंट उपयोग करें, आप जिस मॉडल आईडी को इंगित करते हैं वह **`claudinio`** है।
    बेस URL **`https://api.claudin.io`** है।
