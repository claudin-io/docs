# FAQ

## Claudin.io بالکل کیا ہے؟

AI کوڈنگ ایجنٹس کے لیے ایک API پراکسی۔ آپ ایک فلیٹ ماہانہ سبسکرپشن ادا کرتے ہیں اور
ایک OpenAI/Anthropic-مطابق API کلید حاصل کرتے ہیں جسے آپ Claude Code, Kilo, Zed,
Codex, Cursor, یا کسی بھی OpenAI کلائنٹ میں استعمال کر سکتے ہیں۔ فی ٹوکن بلنگ نہیں۔

## کیا یہ واقعی لامحدود ہے؟

استعمال لامحدود ہے — کوئی درخواست کاؤنٹر یا ٹوکن میٹر نہیں۔ صرف ایک حد ہے
**خرچ سے تحفظ کی حد** فی وقت کی ونڈو جو بھاگتے ہوئے ایجنٹ کو آپ کے پلان کو ختم کرنے سے روکتی ہے۔
عام انٹرایکٹو کام میں آپ شاذ و نادر ہی اسے چھوتے ہیں۔ دیکھیں
[پلانز اور حدود](plans.md)۔

## میں کون سا ماڈل استعمال کروں؟

ہمیشہ **`claudinio`** (یا `claudinio/claudinio` ان کلائنٹس کے لیے جو
`provider/model` فارم چاہتے ہیں)۔ بیس URL ہے `https://api.claudin.io`۔

## کیا میں `Authorization` یا `x-api-key` سے تصدیق کروں؟

دونوں کام کرتے ہیں۔ `Authorization: Bearer YOUR_API_KEY` یا `x-api-key: YOUR_API_KEY`۔

## کیا میں اسے کسی ایسے ٹول کے ساتھ استعمال کر سکتا ہوں جو فہرست میں نہیں ہے؟

ہاں — کوئی بھی ٹول جو آپ کو ایک کسٹم OpenAI بیس URL سیٹ کرنے دیتا ہے وہ کام کرتا ہے۔ استعمال کریں
[عام OpenAI سیٹ اپ](clients/openai-compatible.md)۔

## کیا یہ ٹول / فنکشن کالنگ کو سپورٹ کرتا ہے؟

ہاں۔ اسی لیے یہ ایجنٹک ایڈیٹرز کے اندر کام کرتا ہے۔ `tools` پاس کریں اور
`tool_calls` پڑھیں جیسے OpenAI API کے ساتھ۔

## کیا یہ تصاویر، آڈیو، یا ویڈیو ہینڈل کر سکتا ہے؟

ہاں، شفاف طریقے سے۔ معیاری OpenAI content blocks بھیجیں؛ پراکسی
تصاویر/آڈیو/ویڈیو کو ماڈل کے دیکھنے سے پہلے متن کی تفصیل یا ٹرانسکرپشن میں تبدیل کر دیتی ہے۔
کنفیگر کرنے کے لیے کچھ خاص نہیں۔

## ونڈو کا سیاق و سباق کتنا ہے؟

256K ٹوکنز۔

## میں کیسے اپ گریڈ یا منسوخ کروں؟

اپنے [ڈیش بورڈ](https://claudin.io/dashboard) سے۔ اپ گریڈ فوری طور پر لاگو ہوتے ہیں
(Stripe کے ذریعے)۔ اگر آپ منسوخ کرتے ہیں، تو آپ اپنا ادا شدہ پلان اس مدت کے اختتام تک رکھتے ہیں
جس کے لیے آپ نے پہلے ہی ادائیگی کی ہے، پھر خود بخود Free پر آ جاتے ہیں۔

## مجھے بجٹ کی خرابی آئی۔ اب کیا؟

آپ موجودہ ونڈو کی خرچ سے تحفظ کی حد تک پہنچ گئے۔ یا تو ونڈو کے ری سیٹ ہونے کا انتظار کریں
(آپ کا ڈیش بورڈ دکھاتا ہے کہ کب) یا بڑی حد کے لیے [اپ گریڈ](plans.md) کریں۔

## ایک درخواست 401 کے ساتھ ناکام ہوئی۔

آپ کی کلید غائب یا غلط ہے۔ ڈیش بورڈ سے دوبارہ کاپی کریں اور یقینی بنائیں کہ
کوئی اضافی وائٹ اسپیس نہیں ہے، اور یہ کہ auth ہیڈر سیٹ ہے۔

## میری کلید لیک ہو گئی۔ میں کیا کروں؟

اسے ڈیش بورڈ سے منسوخ کریں اور فوری طور پر ایک نئی جنریٹ کریں۔ کلیدوں کو
پاس ورڈز کی طرح سمجھیں — انہیں کبھی کمٹ نہ کریں یا عوامی طور پر شیئر نہ کریں۔

## مجھے مدد کہاں سے ملے گی؟

اپنے [ڈیش بورڈ](https://claudin.io/dashboard) میں **Support** کارڈ سے ٹکٹ کھولیں،
یا support@claudin.io پر ای میل کریں۔ ہم آپ سے رابطہ کریں گے۔
