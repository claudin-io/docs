# FAQ

## Claudin.ioとは正確には何ですか？

AIコーディングエージェント向けのAPIプロキシです。定額の月額サブスクリプションを支払うことで、OpenAI/Anthropic互換のAPIキーを入手でき、Claude Code、Kilo、Zed、Codex、Cursor、または任意のOpenAIクライアントで使用できます。トークンごとの課金はありません。

## 本当に無制限ですか？

使用量は無制限です — リクエストカウンターやトークンメーターはありません。唯一の制限は、時間枠ごとの**支出保護上限**で、暴走したエージェントがあなたのプランを枯渇させるのを防ぎます。通常の対話作業では、これに達することはほとんどありません。[プランと制限](plans.md)を参照してください。

## どのモデルを使用すればよいですか？

常に **`claudinio`** (または `provider/model` 形式を必要とするクライアント向けに `claudinio/claudinio`) を使用してください。ベースURLは `https://api.claudin.io` です。

## 認証は `Authorization` と `x-api-key` のどちらですか？

どちらでも機能します。`Authorization: Bearer YOUR_API_KEY` または `x-api-key: YOUR_API_KEY` です。

## リストにないツールでも使用できますか？

はい — カスタムOpenAIベースURLを設定できる任意のツールで動作します。[汎用OpenAI設定](clients/openai-compatible.md)を参照してください。

## ツール/関数呼び出しをサポートしていますか？

はい。それがエージェンティックエディター内で動作する理由です。OpenAI API と同様に `tools` を渡し、`tool_calls` を読み取ります。

## 画像、音声、動画を処理できますか？

はい、透過的に処理します。標準のOpenAIコンテンツブロックを送信すると、プロキシが画像/音声/動画をテキスト説明または文字起こしに変換してからモデルに渡します。特別な設定は必要ありません。

## コンテキストウィンドウは？

256K トークンです。

## アップグレードやキャンセルはどうすればよいですか？

[ダッシュボード](https://claudin.io/dashboard)から行えます。アップグレードは即時適用されます（Stripe経由）。キャンセルした場合、既に支払い済みの期間の終了までは有料プランが継続され、その後自動的に無料プランにダウングレードされます。

## 予算エラーが発生しました。どうすれば？

現在のウィンドウの支出保護上限に達しました。ウィンドウがリセットされるのを待つか（ダッシュボードにタイミングが表示されます）、[アップグレード](plans.md)して上限を大きくしてください。

## リクエストが401エラーで失敗しました。

キーが欠けているか間違っています。ダッシュボードからキーを再コピーし、余分な空白がなく、認証ヘッダーが設定されていることを確認してください。

## キーが漏洩しました。どうすれば？

ダッシュボードからキーを直ちに取り消し、新しいキーを生成してください。キーはパスワードと同様に扱い、コミットしたり公開したりしないでください。

## サポートはどこで受けられますか？

[ダッシュボード](https://claudin.io/dashboard)の **サポート** カードからチケットを開くか、サポートにメールしてください。折り返しご連絡します。
