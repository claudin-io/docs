## 为什么选择 Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __无限使用__

    ---

    没有请求计数，没有 token 计量。唯一的防护措施是每个周期设置一个 **消费保护** 上限，防止代理失控——在正常工作中几乎不会触发。

-   :material-power-plug:{ .lg .middle } __与您的工具兼容__

    ---

    与 OpenAI 和 Anthropic 兼容的端点。只需几分钟即可将其集成到 OpenCode、Claude Code、Kilo Code、Zed、Codex、Cursor 或任何 OpenAI 客户端中。

-   :material-cash-multiple:{ .lg .middle } __可预测的成本__

    ---

    固定订阅。月底没有意外账单，没有“这次重构要花多少钱？”的焦虑。

-   :material-shield-check:{ .lg .middle } __内置消费保护__

    ---

    逐级递增的每周期预算上限可防止无限循环的代理耗尽您的套餐。透明且实时显示在您的仪表盘上。

</div>

## 四步上手

1. **[创建账户](getting-started/account.md)** 并复制您的 API 密钥。
2. **[设置您的 API 密钥](getting-started/set-your-key.md)**，仅需在 shell 中设置一次。
3. **[连接您的工具](clients/opencode.md)**——选择您的编辑器或代理。
4. **[发起首次调用](getting-started/first-call.md)** 并开始构建。

!!! tip "模型名称始终为 `claudinio`"
    无论您使用什么客户端，您指向的模型 ID 都是 **`claudinio`**。
    基础 URL 是 **`https://api.claudin.io`**。
