# FAQ

## Claudin.io는 정확히 무엇인가요?

AI 코딩 에이전트를 위한 API 프록시입니다. 고정 월 구독료를 지불하면 OpenAI/Anthropic 호환 API 키를 받아 Claude Code, Kilo, Zed, Codex, Cursor 또는 모든 OpenAI 클라이언트에 사용할 수 있습니다. 토큰당 요금은 없습니다.

## 정말 무제한인가요?

사용량은 무제한입니다 — 요청 카운터나 토큰 측정기가 없습니다. 유일한 제한은 시간 창당 **지출 보호 한도(spend-protection cap)**로, 통제 불능 에이전트가 요금제를 소진하는 것을 막습니다. 일반적인 대화형 작업에서는 거의 도달하지 않습니다. [요금제 및 제한](plans.md)을 참조하세요.

## 어떤 모델을 사용하나요?

항상 **`claudinio`** (또는 `provider/model` 형식을 원하는 클라이언트의 경우 `claudinio/claudinio`)를 사용하세요. 기본 URL은 `https://api.claudin.io`입니다.

## `Authorization` 또는 `x-api-key`로 인증하나요?

둘 다 가능합니다. `Authorization: Bearer YOUR_API_KEY` 또는 `x-api-key: YOUR_API_KEY`를 사용하세요.

## 목록에 없는 도구와도 사용할 수 있나요?

예 — 사용자 정의 OpenAI 기본 URL을 설정할 수 있는 모든 도구에서 작동합니다. [일반 OpenAI 설정](clients/openai-compatible.md)을 사용하세요.

## 도구/함수 호출을 지원하나요?

예. 그래서 에이전트 편집기 내에서 작동합니다. OpenAI API와 마찬가지로 `tools`를 전달하고 `tool_calls`를 읽으세요.

## 이미지, 오디오 또는 비디오를 처리할 수 있나요?

예, 투명하게 처리합니다. 표준 OpenAI 콘텐츠 블록을 보내면 프록시가 모델이 보기 전에 이미지/오디오/비디오를 텍스트 설명 또는 전사로 변환합니다. 특별히 구성할 필요가 없습니다.

## 컨텍스트 창은 어떻게 되나요?

256K 토큰입니다.

## 업그레이드하거나 취소하려면 어떻게 하나요?

[대시보드](https://claudin.io/dashboard)에서 가능합니다. 업그레이드는 즉시 적용됩니다(Stripe를 통해). 취소하는 경우 이미 지불한 기간이 끝날 때까지 유료 요금제가 유지되며, 그 후 자동으로 무료 요금제로 전환됩니다.

## 예산 오류가 발생했습니다. 어떻게 해야 하나요?

현재 창의 지출 보호 한도에 도달했습니다. 창이 재설정될 때까지 기다리거나(대시보드에 시기가 표시됨) 더 큰 한도를 위해 [업그레이드](plans.md)하세요.

## 요청이 401 오류로 실패했습니다.

키가 누락되었거나 잘못되었습니다. 대시보드에서 키를 다시 복사하고 불필요한 공백이 없으며 인증 헤더가 설정되어 있는지 확인하세요.

## 키가 유출되었습니다. 어떻게 해야 하나요?

대시보드에서 즉시 키를 취소하고 새 키를 생성하세요. 키를 비밀번호처럼 취급하세요 — 절대 커밋하거나 공개적으로 공유하지 마세요.

## 어디에서 도움을 받을 수 있나요?

[대시보드](https://claudin.io/dashboard)의 **지원(Support)** 카드에서 티켓을 열거나 support로 이메일을 보내세요. 답변을 드리겠습니다.
