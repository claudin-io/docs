# কার্সর

[কার্সর](https://cursor.com) আপনাকে এর সেটিংসের মাধ্যমে একটি OpenAI-সামঞ্জস্যপূর্ণ মডেল যোগ করতে দেয়। Claudin.io OpenAI বেস URL ওভাররাইডের মাধ্যমে সংযুক্ত হয়।

## সেটআপ

1. **কার্সর → সেটিংস → মডেল** (অথবা **কার্সর সেটিংস → AI**) খুলুন।
2. **OpenAI API কী**-তে স্ক্রোল করুন এবং **OpenAI বেস URL ওভাররাইড** অপশনটি প্রসারিত করুন।
3. সেট করুন:

    | ফিল্ড | মান |
    | --- | --- |
    | OpenAI API কী | `YOUR_API_KEY` |
    | বেস URL | `https://api.claudin.io/v1` |

4. **মডেল**-এর অধীনে, একটি কাস্টম মডেল যোগ করুন যার নাম **`claudinio`** এবং এটি সক্রিয় করুন।
5. আপনি যদি চান যে কার্সর একচেটিয়াভাবে Claudin.io ব্যবহার করুক, তাহলে অন্যান্য ডিফল্ট মডেলগুলি নিষ্ক্রিয় করুন।

!!! note "কার্সরের নিজস্ব বৈশিষ্ট্য"
    কার্সরের এজেন্টিক বৈশিষ্ট্যগুলি একটি OpenAI-সামঞ্জস্যপূর্ণ চ্যাট মডেলের সাথে সবচেয়ে ভালো কাজ করে। `claudinio` টুল কল সমর্থন করে, তাই কম্পোজার/এজেন্ট ফ্লো কাজ করে। কিছু কার্সর-মালিকানাধীন বৈশিষ্ট্য (ট্যাব অটো-কমপ্লিট, ইত্যাদি) কার্সরের নিজস্ব মডেলে চলে এবং আপনার প্রদানকারী ওভাররাইডের মাধ্যমে রুট করা হয় না।

## যাচাই করুন

কার্সরে একটি চ্যাট খুলুন, **claudinio** নির্বাচন করুন এবং একটি বার্তা পাঠান। যদি আপনি উত্তর পান, তাহলে সব ঠিক আছে। যদি না পান, তাহলে নিশ্চিত করুন যে বেস URL `/v1` দিয়ে শেষ হয়েছে এবং কীটি অতিরিক্ত স্পেস ছাড়া পেস্ট করা হয়েছে।

| সেটিং | মান |
| --- | --- |
| বেস URL | `https://api.claudin.io/v1` |
| মডেল | `claudinio` |
