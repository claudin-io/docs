# Cursor

[Cursor](https://cursor.com) आपको अपनी सेटिंग्स के माध्यम से एक OpenAI-संगत मॉडल जोड़ने देता है। Claudin.io OpenAI बेस URL ओवरराइड के माध्यम से प्लग इन होता है।

## सेटअप

1. **Cursor → Settings → Models** (या **Cursor Settings → AI**) खोलें।
2. **OpenAI API Key** पर स्क्रॉल करें और **Override OpenAI Base URL** विकल्प का विस्तार करें।
3. सेट करें:

    | फ़ील्ड | मान |
    | --- | --- |
    | OpenAI API Key | `YOUR_API_KEY` |
    | Base URL | `https://api.claudin.io/v1` |

4. **Models** के अंतर्गत, **`claudinio`** नाम का एक कस्टम मॉडल जोड़ें और इसे सक्षम करें।
5. यदि आप चाहते हैं कि Cursor केवल Claudin.io का उपयोग करे तो अन्य डिफ़ॉल्ट मॉडल को अक्षम करें।

!!! note "Cursor की अपनी विशेषताएं"
    Cursor की एजेंटिक विशेषताएं एक OpenAI-संगत चैट मॉडल के साथ सबसे अच्छा काम करती हैं। `claudinio` टूल कॉल का समर्थन करता है, इसलिए Composer/Agent फ़्लो काम करते हैं। कुछ Cursor-मालिकाना विशेषताएं (Tab ऑटोकम्प्लीट, आदि) Cursor के अपने मॉडल पर चलती हैं और आपके प्रदाता ओवरराइड के माध्यम से रूट नहीं होती हैं।

## सत्यापित करें

Cursor में एक चैट खोलें, **claudinio** चुनें, और एक संदेश भेजें। यदि आपको उत्तर मिलता है, तो आप सेट हैं। यदि नहीं, तो दोबारा जांचें कि बेस URL `/v1` में समाप्त होता है और कुंजी बिना अतिरिक्त स्थानों के चिपकाई गई है।

| सेटिंग | मान |
| --- | --- |
| Base URL | `https://api.claudin.io/v1` |
| Model | `claudinio` |
