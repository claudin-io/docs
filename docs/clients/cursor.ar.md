# المؤشر

[المؤشر](https://cursor.com) يتيح لك إضافة نموذج متوافق مع OpenAI من خلال إعداداته.
يتصل Claudin.io عبر تجاوز عنوان URL الأساسي لـ OpenAI.

## الإعداد

1. افتح **المؤشر → الإعدادات → النماذج** (أو **إعدادات المؤشر → الذكاء الاصطناعي**).
2. انتقل إلى **مفتاح OpenAI API** وقم بتوسيع خيار **تجاوز عنوان URL الأساسي لـ OpenAI**.
3. قم بتعيين:

    | الحقل | القيمة |
    | --- | --- |
    | مفتاح OpenAI API | `YOUR_API_KEY` |
    | عنوان URL الأساسي | `https://api.claudin.io/v1` |

4. ضمن **النماذج**، أضف نموذجًا مخصصًا باسم **`claudinio`** وقم بتمكينه.
5. قم بتعطيل النماذج الافتراضية الأخرى إذا كنت تريد أن يستخدم المؤشر Claudin.io حصريًا.

!!! note "ميزات المؤشر الخاصة"
    تعمل ميزات المؤشر الوكيلية بشكل أفضل مع نموذج محادثة متوافق مع OpenAI.
    يدعم `claudinio` استدعاءات الأدوات، لذا فإن تدفقات Composer/Agent تعمل. بعض الميزات الخاصة بالمؤشر (الإكمال التلقائي للعلامات، إلخ) تعمل على نماذج المؤشر الخاصة ولا يتم توجيهها عبر تجاوز المزود الخاص بك.

## التحقق

افتح دردشة في المؤشر، واختر **claudinio**، وأرسل رسالة. إذا تلقيت ردًا، فأنت جاهز. إذا لم يكن الأمر كذلك، فتأكد من أن عنوان URL الأساسي ينتهي بـ `/v1` وأن المفتاح تم لصقه بدون مسافات إضافية.

| الإعداد | القيمة |
| --- | --- |
| عنوان URL الأساسي | `https://api.claudin.io/v1` |
| النموذج | `claudinio` |
