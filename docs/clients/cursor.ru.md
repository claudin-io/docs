# Cursor

[Cursor](https://cursor.com) позволяет добавить совместимую с OpenAI модель через настройки. Claudin.io подключается через переопределение базового URL OpenAI.

## Настройка

1. Откройте **Cursor → Настройки → Модели** (или **Настройки Cursor → ИИ**).
2. Прокрутите до **OpenAI API Key** и разверните опцию **Override OpenAI Base URL**.
3. Укажите:

    | Поле | Значение |
    | --- | --- |
    | OpenAI API Key | `YOUR_API_KEY` |
    | Base URL | `https://api.claudin.io/v1` |

4. В разделе **Models** добавьте кастомную модель с именем **`claudinio`** и включите её.
5. Отключите остальные стандартные модели, если хотите, чтобы Cursor использовал только Claudin.io.

!!! note "Возможности Cursor"
    Агентные возможности Cursor лучше всего работают с совместимой с OpenAI чат-моделью. `claudinio` поддерживает вызовы инструментов, поэтому потоки Composer/Agent работают. Некоторые собственные функции Cursor (автодополнение Tab и т.д.) выполняются на собственных моделях Cursor и не направляются через ваше переопределение провайдера.

## Проверка

Откройте чат в Cursor, выберите **claudinio** и отправьте сообщение. Если получите ответ, всё готово. Если нет, перепроверьте, что base URL заканчивается на `/v1` и ключ вставлен без лишних пробелов.

| Настройка | Значение |
| --- | --- |
| Base URL | `https://api.claudin.io/v1` |
| Model | `claudinio` |
