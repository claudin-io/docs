# Cursor

[Cursor](https://cursor.com) آپ کو اپنی ترتیبات کے ذریعے ایک OpenAI-مطابق ماڈل شامل کرنے دیتا ہے۔ Claudin.io OpenAI بیس URL اوور رائڈ کے ذریعے پلگ ان ہوتا ہے۔

## سیٹ اپ

1. **Cursor → Settings → Models** (یا **Cursor Settings → AI**) کھولیں۔
2. **OpenAI API Key** تک اسکرول کریں اور **Override OpenAI Base URL** آپشن کو پھیلائیں۔
3. درج ذیل سیٹ کریں:

    | فیلڈ | ویلیو |
    | --- | --- |
    | OpenAI API کلید | `YOUR_API_KEY` |
    | بنیادی URL | `https://api.claudin.io/v1` |

4. **Models** کے تحت، ایک حسب ضرورت ماڈل جس کا نام **`claudinio`** ہے شامل کریں اور اسے فعال کریں۔
5. اگر آپ چاہتے ہیں کہ Cursor صرف Claudin.io استعمال کرے تو دیگر ڈیفالٹ ماڈلز کو غیر فعال کر دیں۔

!!! note "Cursor کی اپنی خصوصیات"
    Cursor کی ایجنٹک خصوصیات OpenAI-مطابق چیٹ ماڈل کے ساتھ بہترین کام کرتی ہیں۔ `claudinio` ٹول کالز کو سپورٹ کرتا ہے، لہٰذا Composer/Agent کے فلو کام کرتے ہیں۔ کچھ Cursor کی ملکیتی خصوصیات (Tab آٹوکمپلیٹ وغیرہ) Cursor کے اپنے ماڈلز پر چلتی ہیں اور آپ کے پرووائیڈر اوور رائڈ کے ذریعے روٹ نہیں ہوتیں۔

## تصدیق

Cursor میں ایک چیٹ کھولیں، **claudinio** کو منتخب کریں اور ایک پیغام بھیجیں۔ اگر آپ کو جواب ملتا ہے تو آپ تیار ہیں۔ اگر نہیں، تو یقینی بنائیں کہ بیس URL `/v1` پر ختم ہوتا ہے اور کلید بغیر اضافی اسپیس کے پیسٹ کی گئی ہے۔

| ترتیب | ویلیو |
| --- | --- |
| بنیادی URL | `https://api.claudin.io/v1` |
| ماڈل | `claudinio` |
