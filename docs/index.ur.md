# لا محدود AI، ایک مقررہ قیمت

Claudin.io ایک API پراکسی ہے AI کوڈنگ ایجنٹس کے لیے۔ ایک **فلیٹ ماہانہ سبسکرپشن** ادا کریں اور جتنا چاہیں استعمال کریں — کوئی فی ٹوکن بلنگ نہیں، کوئی درخواست کاؤنٹر نہیں، کوئی کریڈٹ جو ختم ہو جائے۔

## کیوں Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __لا محدود استعمال__

    ---

    کوئی درخواست کاؤنٹ نہیں، کوئی ٹوکن میٹرنگ نہیں۔ واحد محافظ ایک فی پیریڈ **اخراجات کی حفاظت** کی حد ہے جو بھاگے ہوئے ایجنٹ کو روکتی ہے — آپ عام کام میں شاید ہی اسے چھوتے ہیں۔

-   :material-power-plug:{ .lg .middle } __آپ کے ٹولز کے ساتھ کام کرتا ہے__

    ---

    OpenAI اور Anthropic کے ساتھ ہم آہنگ اینڈ پوائنٹس۔ اسے OpenCode، Claude Code، Kilo Code، Zed، Codex، Cursor، یا کسی بھی OpenAI کلائنٹ میں منٹوں میں ڈالیں۔

-   :material-cash-multiple:{ .lg .middle } __پیش قیاس لاگت__

    ---

    ایک مقررہ سبسکرپشن۔ مہینے کے آخر میں کوئی حیرت انگیز بل نہیں، کوئی 'اس ریفیکٹر کی قیمت کتنی ہوگی؟' کی پریشانی نہیں۔

-   :material-shield-check:{ .lg .middle } __خرچ کی حفاظت شامل__

    ---

    ایک کاسکیڈنگ فی پیریڈ بجٹ کی حد آپ کو لامحدود لوپ ایجنٹس سے بچاتی ہے جو آپ کا منصوبہ جلا دیتے ہیں۔ شفاف اور آپ کے ڈیش بورڈ میں لائیو دکھایا گیا ہے۔

</div>

## 4 مراحل میں شروع کریں

1. **[ایک اکاؤنٹ بنائیں](getting-started/account.md)** اور اپنی API کلید کاپی کریں۔
2. **[اپنی API کلید سیٹ کریں](getting-started/set-your-key.md)** ایک بار اپنے شیل میں۔
3. **[اپنا ٹول منسلک کریں](clients/opencode.md)** — اپنا ایڈیٹر یا ایجنٹ منتخب کریں۔
4. **[اپنی پہلی کال کریں](getting-started/first-call.md)** اور تعمیر شروع کریں۔

!!! tip "ماڈل کا نام ہمیشہ `claudinio` ہے"
    آپ جو بھی کلائنٹ استعمال کریں، ماڈل آئی ڈی جس کی طرف اشارہ کرتے ہیں وہ **`claudinio`** ہے۔ بیس URL **`https://api.claudin.io`** ہے۔
