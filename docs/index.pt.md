# IA Ilimitada, um preço fixo

Claudin.io é um proxy de API para agentes de codificação com IA. Paga uma **subscrição mensal fixa** e usa-o tanto quanto quiser — sem faturação por token, sem contadores de pedidos, sem créditos que expiram.

## Porquê Claudin.io

<div class="grid cards" markdown>

-   :material-infinity:{ .lg .middle } __Utilização ilimitada__

    ---

    Sem contagem de pedidos, sem medição de tokens. A única salvaguarda é um limite de **proteção de gastos** por período que pára um agente descontrolado — quase nunca o atinges num trabalho normal.

-   :material-power-plug:{ .lg .middle } __Funciona com as tuas ferramentas__

    ---

    Endpoints compatíveis com OpenAI e Anthropic. Integra-o no OpenCode, Claude Code, Kilo Code, Zed, Codex, Cursor ou em qualquer cliente OpenAI em minutos.

-   :material-cash-multiple:{ .lg .middle } __Custo previsível__

    ---

    Uma subscrição fixa. Sem facturas surpresa no final do mês, sem ansiedade do "quanto vai custar esta refatoração?".

-   :material-shield-check:{ .lg .middle } __Proteção de gastos incorporada__

    ---

    Um limite orçamental por período em cascata protege-te de agentes em loop infinito a queimarem o teu plano. Transparente e mostrado em tempo real no teu painel de controlo.

</div>

## Começa em 4 passos

1. **[Cria uma conta](getting-started/account.md)** e copia a tua chave de API.
2. **[Define a tua chave de API](getting-started/set-your-key.md)** uma vez no teu terminal.
3. **[Conecta a tua ferramenta](clients/opencode.md)** — escolhe o teu editor ou agente.
4. **[Faz a tua primeira chamada](getting-started/first-call.md)** e começa a construir.

!!! tip "O nome do modelo é sempre `claudinio`"
    Seja qual for o cliente que uses, o ID do modelo para o qual apontas é **`claudinio`**.
    O URL base é **`https://api.claudin.io`**.
